# DevOps

::: info 简介
本栏目收录 DevOps 和运维相关的技术文档，包括容器化、自动化运维、CI/CD 等内容。
:::

## 📖 技术概述

DevOps 是开发（Development）和运维（Operations）的结合，强调通过自动化工具和流程来提高软件交付的速度和质量。DevOps 文化促进了持续集成、持续交付（CI/CD）、基础设施即代码（IaC）等最佳实践的落地。

### 核心技术栈

- **容器化技术**：Docker 提供轻量级应用隔离，Kubernetes 实现容器编排
- **配置管理**：自动化部署和配置管理工具
- **CI/CD 流水线**：GitLab CI、Jenkins 等持续集成工具
- **监控与日志**：ELK 栈（Elasticsearch、Logstash、Kibana）日志分析
- **基础设施**：Linux 系统管理、Nginx 负载均衡和反向代理
