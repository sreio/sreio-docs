# 编程语言

::: info 简介
本栏目收录各类编程语言的学习资料，包括语法、框架、设计模式等内容。
:::

## 📖 技术概述

编程语言是软件开发的基础工具，不同的语言有各自的特点和适用场景。选择合适的编程语言对于项目的成功至关重要。

### 主要编程语言

- **Golang（Go）**：Google 开发的现代编程语言，以简洁、高效和并发友好著称。特别适合构建高性能的网络服务、分布式系统和云原生应用。内置的 goroutine 和 channel 让并发编程变得简单优雅。

- **PHP**：Web 开发的经典语言，广泛应用于服务器端开发。现代 PHP 框架如 Laravel、Symfony 提供了优雅的开发体验。PHP 8+ 引入了 JIT 编译器和诸多新特性，性能和开发效率都有显著提升。
