::: info 简介

## 📖 技术概述

人工智能（Artificial Intelligence, AI）是计算机科学的一个重要分支，旨在创建能够模拟人类智能行为的系统。近年来，随着深度学习技术的突破，AI 在计算机视觉、自然语言处理、语音识别等领域取得了革命性进展。

### 核心技术领域

- **机器学习（ML）**：通过算法使计算机从数据中学习规律
- **深度学习（DL）**：基于神经网络的多层学习方法
- **大语言模型（LLM）**：如 GPT、Claude 等新一代 AI 模型
- **Transformer**：现代 AI 的核心架构，彻底改变了 NLP 领域
