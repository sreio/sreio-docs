# 基础知识

::: info 简介
本栏目收录计算机基础知识相关文档，包括算法、网络、系统等核心内容。
:::

## 📖 技术概述

计算机基础知识是每个软件工程师必须掌握的核心内容。扎实的基础知识不仅有助于编写高效的代码，还能帮助理解系统底层原理，在面对复杂问题时能够做出更好的技术决策。

### 核心知识领域

- **数据结构与算法**
  - 数据结构：数组、链表、树、图、哈希表等基础结构
  - 算法设计：排序、搜索、动态规划、贪心算法、分治算法等
  - 算法复杂度：时间复杂度和空间复杂度分析

- **计算机网络**
  - **TCP/IP 协议栈**：网络通信的基础协议
  - **HTTP/HTTPS**：Web 应用的核心协议
  - **网络安全**：加密、认证、防火墙等安全机制

### 学习资源

- [LeetCode](https://leetcode.com/) - 算法练习平台
- [算法可视化](https://visualgo.net/) - 数据结构和算法可视化
- [MDN Web Docs - HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP) - HTTP 协议详解
- [RFC 文档](https://www.rfc-editor.org/) - 网络协议标准文档
- [TCP/IP 详解](https://www.ietf.org/) - IETF 官方网站
